❓ Dúvidas frequentes

Perguntas frequentes

Tudo que você precisa saber antes de começar.

O MEI Control Pro é realmente gratuito para sempre?

+

Sim! O plano Gratuito é permanente e inclui dashboard financeiro, até 20 lançamentos por mês, perfil do MEI e controle do limite anual. Você só faz upgrade se quiser recursos avançados como relatórios, DAS e IRPF.

O MEI Control Pro substitui o contador?

+

O MEI Control Pro é um sistema de gestão financeira para controle do dia a dia do MEI. Para obrigações contábeis mais complexas ou em caso de dúvidas fiscais, recomendamos sempre consultar um contador. Mas para a gestão cotidiana de receitas, despesas e DAS, o MEI Control Pro cobre tudo.

Posso cancelar minha assinatura a qualquer momento?

+

Sim, absolutamente. Sem multa, sem fidelidade, sem burocracia. Cada pagamento ativa o plano por 30 dias. Se você não renovar, sua conta volta automaticamente para o plano Gratuito — seus dados são preservados por 30 dias.

Meus dados financeiros ficam seguros?

+

Sim. Utilizamos o Supabase como banco de dados com criptografia de ponta a ponta, autenticação segura e backups automáticos. Seus dados nunca são compartilhados com terceiros.

O MEI Control Pro funciona para qualquer tipo de MEI?

+

Sim! O sistema foi desenvolvido para todos os ramos de atividade permitidos ao MEI — comércio, serviços, industria artesanal, transporte, alimentação, tecnologia e muito mais. As categorias são personalizáveis para o seu negócio.

Como funciona o pagamento? É automático?

+

Não. Cada pagamento é avulso e manual, via Mercado Pago. Aceitamos cartão de crédito, débito, Pix e boleto bancário. Não há cobrança recorrente automática — você decide quando renovar.

Preciso instalar algum aplicativo?

+

Não! O MEI Control Pro é 100% web. Acesse pelo navegador de qualquer dispositivo — celular, tablet ou computador — sem precisar instalar nada. A interface é totalmente responsiva.

O que acontece se eu ultrapassar o limite do MEI?

+

O MEI Control Pro emite alertas automáticos quando você atingir 80% e 95% do limite anual de R$ 81.000. Isso te dá tempo para se planejar — seja reduzindo vendas, abrindo uma ME ou planejando a transição para o próximo ano fiscal.
